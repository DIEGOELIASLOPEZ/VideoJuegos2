## 🎮 Controls
- **W/A/S/D** → Move  
- **Space** → Jump  
- **Shift** → Sprint  
- **Ctrl** → Crouch  
- **Mouse** → Aim / Shoot  

---

## 🚀 How to Run
1. Clone or download this repository.  
2. Open the project in **Unity 6.1 or later**.  
3. Run the main scene (`MainScene.unity`).  
4. Play and enjoy! 🎮
